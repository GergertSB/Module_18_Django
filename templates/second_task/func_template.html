Во-первых, для облегчения взаимодействия, вот некоторая информация:

  • pip install django - команда установки фреймворка Django
  • django-admin startproject myapp - команда создания проекта Django, где myapp - это произвольное название создаваемого проекта
  • python3 manage.py startapp app - команда создания приложения внутри нашего проекта, где app - это название создаваемого приложения
  • python3 manage.py runserver - это команда запуска сервера Django для начала работы нашего приложения

ВАЖНО: перед запуском сервера нужно находиться в директории нашего проекта (в данном случае в папке myapp).
